# VoterFlow: The Election Navigator

An interactive, high-fidelity React application designed to guide voters through the complex U.S. election process. Built with a focus on premium aesthetics, smooth animations, and clear storytelling.

## 🚀 Features
- **Step-by-Step Guide:** 7 comprehensive sections from Registration to Inauguration.
- **Interactive Animations:** Scroll-triggered card reveals and progress tracking.
- **Electoral Counter:** Animated visualization of the 270 electoral vote threshold.
- **Premium Design:** Dark mode theme with glassmorphism and vibrant gradients.
- **Fully Responsive:** Optimized for all screen sizes.

## 🛠️ Tech Stack
- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Vanilla CSS (Modern custom properties & Flex/Grid)
- **Icons:** Custom SVG components

## 📦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Jha-2022/ElectionExplorer1.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
